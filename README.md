# Descripción del Servicio Innovador

## CodeBoost Sessions

CodeBoost Sessions es un servicio diseñado para aumentar la productividad de estudiantes y profesionales mientras disfrutan de los productos de la cafetería.

### Funcionamiento

1. El usuario escanea un código QR.
2. Selecciona una meta de trabajo o estudio.
3. Activa una sesión Pomodoro de concentración.
4. Obtiene estadísticas y recompensas por completar sesiones.

### Beneficios

* Mayor concentración.
* Mejor gestión del tiempo.
* Experiencia innovadora dentro de la cafetería.
* Integración entre tecnología y bienestar.
